# 麦肯锡经典框架集

## 13. GE-麦肯锡矩阵 (GE-McKinsey Matrix)

**定义**: 战略工具，用于评估和管理公司的业务单元或产品线组合。

**两个维度**:
- **行业吸引力** (Industry Attractiveness): 市场增长率、盈利能力、竞争强度、监管环境
- **竞争优势** (Competitive Strength): 市场份额、品牌实力、产品质量、成本结构

**3x3 矩阵决策**:
| | 竞争优势强 | 竞争优势中 | 竞争优势弱 |
|--|-----------|-----------|-----------|
| **行业吸引力高** | 加大投资 | 选择性投资 | 评估机会 |
| **行业吸引力中** | 选择性投资 | 维持现状 | 考虑退出 |
| **行业吸引力低** | 收割利润 | 考虑退出 | 剥离 |

**使用方法**:
1. 评估每个业务单元所在行业的吸引力
2. 分析每个业务单元的竞争优势
3. 在矩阵中定位每个业务单元
4. 做出投资/维持/剥离的战略决策

## 14. 麦肯锡 7S 框架 (McKinsey 7-S Framework)

**定义**: 分析组织有效性的管理工具，检查七个相互关联的内部要素。

**七个要素**:
- **硬件 (Hard S)**:
  1. **Strategy (战略)**: 长期计划和目标
  2. **Structure (结构)**: 组织架构和部门设置
  3. **Systems (系统)**: 流程、程序和工作流

- **软件 (Soft S)**:
  4. **Shared Values (共同价值观)**: 组织核心文化和信念
  5. **Style (风格)**: 领导风格和管理方式
  6. **Staff (人员)**: 人才、能力和岗位配置
  7. **Skills (技能)**: 个人和组织层面的能力

**核心原则**: 七个要素相互关联，一个要素的变化会影响其他要素。

**使用方法**:
1. 分析每个要素的现状
2. 识别要素之间的不一致
3. 确定需要调整的要素
4. 确保七个要素协调一致

## 15. 业务系统框架 (Business System Framework)

**定义**: 麦肯锡战略工具，帮助公司理解和优化整个业务系统。

**分析维度**:
1. **核心流程**: 生产、交付、服务提供等关键业务活动
2. **资源评估**: 人员、技术、资本、材料的使用效率
3. **价值主张**: 质量、价格、创新、客户体验的差异化
4. **客户互动**: 营销、销售、售后支持的全流程

**使用方法**:
1. 绘制驱动业务的核心流程
2. 评估支撑流程的资源
3. 审视价值主张的竞争力
4. 检查客户互动的每个触点
5. 识别改进机会

## 16. 行业成本曲线 (Industry Cost Curve)

**定义**: 麦肯锡工具，帮助公司了解相对于竞争对手的成本定位。

**分析步骤**:
1. **收集成本数据**: 生产、劳动力、材料、管理费用
2. **绘制成本曲线**: 从低到高排列，绘制行业成本曲线图
3. **分析自身定位**: 确定在曲线上的位置
4. **识别成本驱动因素**: 规模经济、技术、供应链效率
5. **制定成本优化策略**: 优化流程、重新谈判供应商合同、投资技术

**应用**: 成本竞争力分析、定价策略、并购目标评估

## 17. SCP 框架 (Structure-Conduct-Performance)

**定义**: 分析市场结构、企业行为和市场绩效之间关系的框架。

**三个维度**:
- **Structure (结构)**: 竞争者数量、市场集中度、进入壁垒、产品差异化
- **Conduct (行为)**: 定价策略、产品供应、营销策略、创新努力
- **Performance (绩效)**: 盈利能力、效率、创新、消费者福利

**动态反馈循环**: 市场绩效变化 → 企业行为调整 → 市场结构演变

**使用方法**:
1. 分析市场结构特征
2. 评估企业竞争行为
3. 衡量市场绩效结果
4. 识别反馈循环
5. 制定战略决策

## 18. 战略控制地图 (Strategic Control Map)

**定义**: 麦肯锡工具，通过分析市值动态识别最大机会和威胁。

**两个维度**:
- **横轴**: 公司规模 (账面价值)
- **纵轴**: 股东表现 (市净率)

**分析要点**:
1. 收集市值数据
2. 在地图上定位公司
3. 分析位置含义
4. 识别收购目标或被收购风险

**应用**: 并购分析、竞争定位、投资决策
